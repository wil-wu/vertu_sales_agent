# AI销售客服仿真测试指标体系

以下是系统化的评估指标框架，分为6大维度：

---

## 一、拟人化体验指标
| 指标 | 说明 | 评分方式 |
|------|------|----------|
| 语言自然度 | 对话是否流畅、口语化，避免机械感 | 1-5分主观评分 |
| 情感共鸣度 | 能否识别并回应用户情绪 | 情绪识别准确率% |
| 个性一致性 | 品牌人设是否稳定，不前后矛盾 | 人设偏离次数 |
| 幽默/温度感 | 适当使用轻松语气的能力 | 主观评分 |
| 停顿/节奏感 | 回复长度和节奏是否像真人 | 回复字数分布 |

---

## 二、购买意愿驱动指标
| 指标 | 说明 | 评分方式 |
|------|------|----------|
| 需求挖掘率 | 成功识别用户潜在需求的比率 | 命中次数/总对话% |
| 产品推荐精准度 | 推荐产品与需求的匹配程度 | 相关性评分 |
| 购买转化率 | 对话结束后实际产生购买行为 | 转化人数/总人数% |
| 加购/收藏率 | 引导用户加入购物车/收藏 | 行为触发率% |
| 促单成功率 | 限时优惠/稀缺性话术的效果 | A/B测试对比 |

---

## 三、问题解决能力指标
| 指标 | 说明 |
|------|------|
| 首次解决率 (FCR) | 一次对话内解决问题的比率 |
| 意图识别准确率 | 正确理解用户问题的比率 |
| 知识覆盖率 | 产品知识库回答覆盖度 |
| 拒识率 / 兜底率 | 无法回答而转人工的频率 |
| 多轮理解能力 | 跨轮次保持上下文的能力 |

---

## 四、销售话术质量指标
| 指标 | 说明 |
|------|------|
| FAB结构完整度 | 特性→优势→利益的表达完整性 |
| 异议处理成功率 | 对价格/质量异议的化解率 |
| 交叉销售触发率 | 主动推荐关联产品的频率 |
| 话术合规率 | 是否存在违规承诺/夸大宣传 |
| 个性化表达率 | 根据用户画像定制话术的比例 |

---

## 五、用户体验指标
| 指标 | 说明 |
|------|------|
| 对话满意度 (CSAT) | 用户主观满意评分 |
| 对话完成率 | 用户主动结束 vs 中途放弃 |
| 平均对话轮次 | 完成任务所需回合数（越少越好）|
| 响应速度感知 | 用户对回复速度的感受 |
| 投诉/负面反馈率 | 触发投诉的对话比例 |

---

## 六、仿真测试专项指标
| 指标 | 说明 |
|------|------|
| 角色扮演还原度 | 模拟真实用户的逼真程度 |
| 边缘场景覆盖率 | 测试用例对异常场景的覆盖 |
| 压力对话稳定性 | 面对投诉/刁难用户的表现 |
| 多轮压测通过率 | 连续多轮对话后的表现衰减 |
| 跨场景泛化能力 | 在新品类/新场景的迁移表现 |

---
